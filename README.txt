== Description ==

Advanced Search Disabler ermöglicht es dir, die Suchfunktion auf deiner WordPress-Website gezielt zu deaktivieren.

## Advanced Search Disabler

Mit dem Advanced Search Disabler kannst du spezifische Einstellungen für die Suchfunktion deiner Website vornehmen.

Du kannst die Suche deaktivieren für:

	* Die gesamte Website
	* Nicht eingeloggte Benutzer
	* oder bestimmte Beiträge ausschließen

Die Einstellungen sind einfach zu konfigurieren und befinden sich unter dem Menüpunkt „Einstellungen → Advanced Search Disabler“.

- Besuche unsere [Website](https://letowp.de/?utm_source=wordpress.org&utm_medium=advanced-search-disabler)
- Oder bewerte uns auf [WordPress](https://wordpress.org/support/plugin/advanced-search-disabler/reviews/?rate=5#new-post)

## Installation

1. Lade das Plugin-Verzeichnis in den `/wp-content/plugins/` Ordner hoch.
2. Aktiviere das Plugin über das 'Plugins' Menü in WordPress.
3. Konfiguriere die Einstellungen im Admin-Bereich unter **Einstellungen -> Advanced Search Disabler**.
